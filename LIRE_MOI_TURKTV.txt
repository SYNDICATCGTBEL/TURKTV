TURKTV - utilisation sur boitier IPTV

Point important:
Un boitier IPTV ne peut pas lancer le script PowerShell de ce PC.
Pour une mise a jour automatique depuis le boitier, il faut entrer une URL web de playlist M3U dans le boitier.

URL a entrer dans le boitier:
https://raw.githubusercontent.com/SYNDICATCGTBEL/TURKTV/main/turktv.m3u

Cette URL pointe vers le fichier turktv.m3u publie sur GitHub.
Le boitier la recharge depuis Internet selon ses propres reglages de rafraichissement.

Fichier prepare dans ce dossier:
turktv.m3u

Pour ouvrir le logiciel visuel depuis le PC:
double-cliquer sur TURKTV_Studio.cmd

Le logiciel permet:
- voir la liste des chaines;
- tester les liens;
- lire une chaine avec VLC integre si VLC est installe;
- rechercher des flux publics turcs;
- extraire des liens m3u8 depuis une page web ou un fichier M3U;
- ajouter les flux selectionnes a turktv.m3u;
- publier sur GitHub.

Ancien menu CMD:
double-cliquer sur Modifier_Chaine_TURKTV.cmd

Le menu permet de modifier:
- le lien video;
- l'image/logo;
- le nom de la chaine si besoin.
- verifier les chaines qui ne fonctionnent pas.
- nettoyer la playlist apres verification.
- importer les chaines absentes de index.m3u.
- tester une chaine avec VLC ou un lecteur HTML.

Apres modification, il faut publier sur GitHub.
Sinon le boitier ne verra pas le changement.
